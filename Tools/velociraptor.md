# 🧰 Velociraptor

Velociraptor is a powerful DFIR tool used for live response and endpoint monitoring.

## 🔥 Features
- Remote artifact collection
- Memory analysis
- Live system investigation
- Endpoint visibility

## ⚡ Use Cases
- Incident response
- Threat hunting
- Malware analysis

## ✅ Advantages
- Fast and scalable
- Works across large environments
- Supports automation

## ⚠️ Why Important
Helps collect volatile and non-volatile data without shutting down systems.
