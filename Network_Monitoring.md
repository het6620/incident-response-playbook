# 🌐 Network Monitoring in Cybersecurity

Network monitoring ensures security, performance, and availability of systems.

---

## 🔍 Types of Network Monitoring

### 1. Performance Monitoring
- Tracks bandwidth, latency
- Tools: Nagios, Zabbix

### 2. Availability Monitoring
- Checks uptime
- Tools: Pingdom

### 3. Security Monitoring
- Detects attacks
- Tools: SIEM, IDS/IPS

### 4. Traffic Monitoring
- NetFlow, sFlow
- Detects anomalies

### 5. Packet Monitoring
- Deep packet inspection
- Tools: Wireshark, tcpdump

### 6. Log Monitoring
- SIEM analysis
- Tools: Splunk, ELK

### 7. Endpoint Monitoring
- Tracks endpoint activity
- Tools: EDR, Sysmon

### 8. Behavioral Monitoring
- Detects anomalies
- Tools: Darktrace

### 9. Signature-Based Monitoring
- Detects known threats
- Tools: Snort, Suricata

---

## 🎯 Importance
- Early attack detection
- Forensics investigation
- Prevent data breaches
